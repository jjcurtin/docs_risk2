39  Focus Blue Mailer

(This SOP is the instructions to fill the mailer envelopes to send to participants so they can get paid)

 

39.0.0.1 Materials

  1. Priority Mail envelope
    • Located in the desktop organizer in room 192
  2. Regular university envelope
    • Located in room 192
    • If out, tell Susan
  3. Stamps
    • If out, tell Susan
  4. STAR mailing labels
    • Located in the desktop organizer in room 192
    • If out: load a sheet of label paper and print to color printer from P:\studydata\risk2\administration\financial\STAR Study Return Address Labels.docx
    • If out of label paper, tell Susan
  5. Focus Blue cards
    • Located in the bottom drawer of the blue cabinet in room 192
  6. Welcome letter
    • Susan will send these to the printer in room 192
  7. W-9 form
    • Located in room 192
    • If we are out: https://www.irs.gov/pub/irs-pdf/fw9.pdf (print 1, page 1 only, 1-sided) then make photocopies using copier code 72463
  8. Highlighter
  9. 2-Page Disclosure
    • Located in room 192
    • If we are out: P:\studydata\risk2\administration\financial\focusblue126784009disclosure.pdf (print 1, then photocopy 2 sided (2-pages front & back total) using copier code 72463 & staple)
  10. Recruitment flyers
    • If out: STAR Google Drive -> UW Letter Size Flyer.pdf (print 2-sided to the color printer on first floor)
    • Small address labels, load a sheet of label paper and print to color printer from P:\studydata\risk2\administration\financial\STAR Study Return Address Labels.docx

39.0.0.2 Procedure

  1. Enter payment card numbers into the log
    • This PC > Private drive > studydata > risk2 > administration > financial > participant_payment_logs > participant_payment_log_risk3
    • If not already added, add subID and date
    • Add cardholder ID
      • Find the ID number on the Focus Blue envelope
      • Last 10 digits of the number on the front of the envelope (starts with 404)
      • Tell Susan to run the script to register the cards
    • On each Focus Blue envelope & Priority Mail Envelope, lightly write the participant’s initials/subid in the corner
  2. Prepare the mailers (Priority Mail envelopes)
    • Each mailer should contain
      • Welcome letter (added at the time of mailing - everything else goes in to pre-make envelopes)
    • Prepaid envelope with the highlighted W-9 form
      • W-9 highlighting instructions
        • Highlight the numbers for items 1, 3, 5, 6
        • Highlight “Social security number”
        • Highlight “Signature of U.S. person”
        • Highlight “date”
      • Put the highlighted form in the smaller envelope
        • Add a stamp
        • Put the STAR address label in the recipient field
        • Add the participant’s initials to the envelope in small writing
      • 2 recruitment flyers
      • Focus Blue card (put the entire envelope in the mailer)
      • 2-page disclosure
  3. Prepare the welcome letters 
    • Open the first payment letter - in P:\studydata\risk2\administration\financial, named “first_payment_2021_04_21.docx”
    • You’ll see a message “opening this document will run the following SQL Command”. Hit No and the document will continue to open.
    • Click the Mailings Menu tab.
    • Click Select Recipients -> use an existing list
    • Browse to the funds file upload .csv that Susan prepared via script.
    • Click Finish & Merge -> Edit individual documents then hit “all”.
    • You’ll usually get a note about an Empty Record, that’s normal.
    • Verify the addresses and names printed correctly for each letter, then print the letters to the First Floor Printer.
  4. Labels are now printed using USPS.com Click’n’Ship after Susan says the upload file is ready (located in P:\studydata\risk2\administration\financial\participant_payment_logs\funds_file_uploads).
    • Login at https://www.usps.com/ : sewanta, password in LastPass
      • Hit Click’n’Ship
      • Scroll down to Step 2, Delivery Address (everything above that is pre-populated and needs no neds).
      • Click Batch Order.
      • Add the address from the focus blue upload file.
      • Beneath the address click “Save to my address book” and then “Add to Batch”.
      • Repeat until all labels are entered. Scroll down and click the button “Select service and packaging”. A new section will appear - you don’t need to make ANY changes. At the bottom click “Add to Cart”
      • On the next page you can review the order and make changes if needed (like if you double entered someone you’ll see that here). If no changes, click Billing Information
      • On the next page you have to click the box “I hereby authorize the USPS to charge….”. Then click the Credit & Debit tab. The lab p-card visa is already pre-populated, you just need to enter 501 for the CVC. Then click Print & Pay
  • To print the labels, you have to scroll down and select all the labels using the checkboxes, then scroll back up and print. Then print the labels to the first floor printer, 2 per page (no receipt needed).
    • Cut them out around the dark border line
    • Tape them to the Priority Mail envelope in the indicated location - be sure NOT to let any tape cover any part of the large barcode!
    • Verify the label name & address against the subid/name on the letter and focus blue card before sealing the Priority Envelope.

39.0.0.3 Lost Card Procedure

  1. Payment card never made it to the participant
    • Go through the steps listed above to send the participant a new packet
  2. Card was lost/stolen/destroyed after activation
    • Do not send replacement card
    • In Google Voice, text the participant the US Bank Card Services number (877-474-0010) and let them know they should contact US Bank directly for a replacement
